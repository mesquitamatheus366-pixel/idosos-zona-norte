# Idosos da Zona Norte

Site da pelada **Idosos da Zona Norte** — futebol mensal e diária, com cadastro de jogadores, sorteio de times equilibrados, resultados e estatísticas.

## Stack
- Vite + React + TypeScript
- TailwindCSS + shadcn/ui
- Supabase (banco + auth)
- Deploy: Vercel

## Como rodar localmente
```bash
npm install
npm run dev
```

## Variáveis
As chaves do Supabase ficam em `utils/supabase/info.tsx` (apenas a `anon key` pública).
